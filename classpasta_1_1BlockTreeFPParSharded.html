Detailed Description

template<std::integral input_type, std::signed_integral size_type, template< typename > typename queue_type = JiffyQueue>
class pasta::BlockTreeFPParSharded< input_type, size_type, queue_type >

A parallel block tree construction algorithm using Rabin-Karp hashes and a sharded hash map.

Template Parameters
input_typeThe type of the characters in the input string
size_typeThe type used for indices etc. (must be a signed integer)
queue_typeThe type of queue to use for communication in the sharded hash map.

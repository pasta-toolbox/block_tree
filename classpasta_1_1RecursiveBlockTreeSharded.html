Detailed Description

template<std::integral input_type, std::signed_integral size_type, uint8_t recursion_level>
class pasta::RecursiveBlockTreeSharded< input_type, size_type, recursion_level >

A parallel block tree construction algorithm using Rabin-Karp hashes and a sharded hash map. Small blocks are not RK-hashed but rather use the blocks themselves.

Template Parameters
input_typeThe type of the characters in the input string
size_typeThe type used for indices etc. (must be a signed integer) in the sharded hash map.

Detailed Description

template<class T, class size_type, uint8_t mersenne_exponent = 0>
class pasta::MersenneRabinKarp< T, size_type, mersenne_exponent >

A Rabin-Karp rolling hasher.

Template Parameters
TThe type of the characters in the text.
size_typeThe type to use for indexing etc.
mersenne_exponentIf using a mersenne prime 2^p-1, then this should be p. If this is 0, a normal modulus operation will be used. Additionally, if this is != 0, then the prime_ attribute will be ignored and '(1 << mersenne_exponent) - 1' will be used instead.
